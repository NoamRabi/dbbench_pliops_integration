# In order to improve dbbench performance we need to enlarge the number of open files the OS is allow us to use,
# please follow the next commands

1. Edit the file /etc/sysctl.conf
    change the value of fs.file-max to 1048576
2. Edit the file /etc/security/limits.conf
    add the following lines at the bottom of the file:
  	* soft     nproc          1048576
		* hard     nproc          1048576
		* soft     nofile         1048576
		* hard     nofile         1048576
		root soft     nproc          1048576
		root hard     nproc          1048576
		root soft     nofile         1048576
		root hard     nofile         1048576   
3. Edit the following file /etc/pam.d/common-session
    add the following line - "session required pam_limits.so"
4. Run the command "echo 65535 /proc/sys/fs/file-max"
5. Edit the files /etc/systemd/user.conf and /etc/systemd/system.conf
    change the value of DefaultLimitNOFILE to 1048576
